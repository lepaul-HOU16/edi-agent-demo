## Learning Paths

### Beginner Path

**Goal**: Learn to build simple queries

1. Read: [User Guide - Introduction](./OSDU_QUERY_BUILDER_USER_GUIDE.md#introduction)
2. Watch: [Video Tutorial 1: Getting Started](./OSDU_QUERY_BUILDER_VIDEO_TUTORIAL_SCRIPT.md#tutorial-1-getting-started-2-minutes)

**Estimated Time**: 30 minutes

### Intermediate Path

**Goal**: Build complex multi-criteria queries

## Quick Reference Cards

### Operators Quick Reference

| Operator | Use For | Example |
|----------|---------|---------|
| `=` | Exact match | `data.operator = "Shell"` |
| `!=` | Not equal | `data.status != "Abandoned"` |
| `>` | Greater than | `data.depth > 3000` |
| `<` | Less than | `data.depth < 5000` |
| `>=` | Greater or equal | `data.depth >= 3000` |
| `<=` | Less or equal | `data.depth <= 5000` |
| `LIKE` | Pattern match | `data.wellName LIKE "North*"` |
| `NOT LIKE` | Exclude pattern | `data.wellName NOT LIKE "*Test*"` |
| `IN` | Multiple values | `data.operator IN ("Shell", "BP")` |
| `NOT IN` | Exclude values | `data.status NOT IN ("Abandoned", "Inactive")` |
| `BETWEEN` | Range | `data.depth BETWEEN 2000 AND 5000` |

### Wildcards Quick Reference

| Wildcard | Matches | Example | Matches |
|----------|---------|---------|---------|
| `*` | Any characters | `North*` | North Sea, Northern, North-15H |
| `?` | Single character | `?ell` | Shell, Well, Bell |
| `*...*` | Contains | `*Test*` | Test-1, MyTest, PreTestPost |

### Keyboard Shortcuts (Desktop)

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Enter` | Execute query |
| `Ctrl/Cmd + N` | Add new criterion |
| `Ctrl/Cmd + H` | Toggle query history |
