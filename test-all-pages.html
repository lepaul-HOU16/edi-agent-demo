üîß Debugging Tips

If you encounter errors:

  1. Console Errors: Open DevTools (F12) ‚Üí Console tab
  2. Network Errors: Open DevTools ‚Üí Network tab ‚Üí Filter by XHR
  3. Authentication Errors: Check if mock token is being sent in Authorization header
  4. Styling Issues: Check if Cloudscape CSS is loaded in globals.css
  5. Component Errors: Check if all imports are correct

Common Fixes:

  • 401 Unauthorized: Backend needs to accept mock tokens
  • CSS not loading: Check globals.css imports Cloudscape CSS first
  • Component not rendering: Check for TypeScript errors
  • API call fails: Check API_BASE_URL in client.ts
